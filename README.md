# Sandip Ghosh - Tabla Maestro Portfolio

This is a premium, high-performance portfolio website built for Tabla Maestro **Sandip Ghosh**, utilizing React 19, Vite, and Tailwind CSS. It features professional Indian classical aesthetic themes, sleek animations, a responsive concert tour tracking component, global collaboration logs, teaching registration forms, and an immersive gallery.

---

## 🚀 How to Host on Vercel (100% Free & Stable)

Vercel is the premier platform for hosting high-speed frontend websites. By hosting with static files in the **`public/assets/`** directory, you do NOT need any database for your images! Placing your images inside the directory is the standard production workflow: it guarantees lightningloader speeds, secure global CDN delivery, and zero database/quota costs.

### Step 1: Prepare Your Image Files Locally
Export this project as a **ZIP file** or **push it to GitHub** via the AI Studio Settings menu. Once you have the files on your computer:
1. Locate the `public/assets/` folder in the project root.
2. Select your high-resolution images.
3. Rename your image files to match the expected names below and save them directly in `public/assets/`.

### Step 2: Push to GitHub & Connect to Vercel
1. Upload your local project folder to a repository on **GitHub** (e.g., `sandip-ghosh-portfolio`).
2. Go to [Vercel](https://vercel.com/) and sign up or log in.
3. Click **"Add New"** > **"Project"** and import your GitHub repository.
4. Since this is a standard Vite React app, Vercel will auto-detect the framework.
5. Click **"Deploy"** — your live portfolio will be online in under 1 minute!

## 🛠️ Local Development & Scripts

To run and view the website locally on your computer, install Node.js and run:

```bash
# Install dependencies
npm install

# Run local development server
npm run dev
```

Your developer workspace will be launched at `http://localhost:3000`.
